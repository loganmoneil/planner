Connect Google Sheets

Your planner data saves directly to a Google Sheet you own — locked to your Google account only.

One-time setup (~8 min):
  1. Create a new Google Sheet at sheets.google.com
  2. In the Sheet, go to Extensions → Apps Script, paste the script (click "Get Script"), then click Deploy → New deployment
  3. Set type to Web App, execute as Me, and set access to Only myself — then deploy and copy the URL
  4. In the same Apps Script project go to Project Settings and note your Script ID (you won't need it here, just confirming the right project)
  5. Go to console.cloud.google.com, create a project, enable the Google Sheets API, then under APIs & Services → Credentials create an OAuth 2.0 Client ID for a Web Application. Add your planner's file URL as an authorized origin. Copy the Client ID.
